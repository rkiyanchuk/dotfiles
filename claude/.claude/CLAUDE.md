# User Notes

Personal notes are stored as Obsidian markdown files at:

```
/Users/ruslan/Library/Mobile Documents/iCloud~md~obsidian/Documents/notes
```

When the user asks to save, store, or add something to their notes,  update or add `.md` notes in that directory (or a relevant subdirectory). Follow Obsidian conventions: use standard markdown, add YAML frontmatter if appropriate (e.g. `tags:`), and use `[[wikilinks]]` for internal links if referencing other notes.
